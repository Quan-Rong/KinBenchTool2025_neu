# 文件归类说明

## 文件归类结果

### 1. MATLAB_Source/ - MATLAB源代码文件夹
包含所有需要重构的MATLAB源代码文件：

- `KnC_Bewertung_alpha20250127.mlapp` - MATLAB App Designer源文件
- `KnC_Bewertung_alpha20250127_exported.m` - 导出的MATLAB代码（10630行，主要重构目标）
- `KinBenchTool_Bump_Plot.m` - Bump测试绘图函数
- `KinBenchTool_Roll_Plot.m` - Roll测试绘图函数
- `KinBenchTool_StaticLoad_Plot.m` - 静态载荷测试绘图函数
- `KinBenchTool_braking_extra_plot.m` - 制动力测试额外绘图函数
- `countlines.pl` - Perl脚本，用于统计文件行数（Windows系统）

**说明**: 这些文件是重构的主要参考，需要1:1转换为Python代码。

### 2. NRAC_G023_Results/ - 数据文件夹（保留）
包含Adams仿真结果数据文件，**需要保留**，因为程序就是用来处理这些数据的：

- `NRAC_G023_parallel_travel.res` - 轮跳测试结果
- `NRAC_G023_roll_angle.res` - 侧倾测试结果
- `NRAC_G023_Lat_static_load.res` - 侧向力测试结果
- `NRAC_G023_Braking_static_load.res` - 制动力测试结果
- `NRAC_G023_Accel_static_load.res` - 加速力测试结果

**说明**: 这些文件是测试数据，重构过程中用于验证Python程序功能。

### 3. Documentation/ - 文档文件夹
包含所有分析文档和计划文档：

- `功能分析文档.md` - 详细的功能分析文档，说明程序的所有功能模块
- `程序说明文档.md` - 程序使用说明和技术文档
- `重构计划.md` - 详细的分模块、分步骤重构计划
- `文件归类说明.md` - 本文件，说明文件归类情况

### 4. Archive/ - 归档文件夹
包含不需要的文件（历史版本、测试文件、资源文件等）：

- `alt_version/` - 历史版本文件夹
- `KnC_Bewertung_alpha04012025_1/` - 打包版本文件夹
- 各种图像文件（.png, .jpg, .jpeg）
- 其他测试文件（.m, .mlapp等）
- `deepseek_test.py` - 测试Python文件

**说明**: 这些文件暂时不需要，但保留作为参考。

### 5. Python_Target/ - Python目标文件夹（待创建）
未来将在此文件夹中创建Python重构版本：

- 项目结构将按照重构计划中的设计创建
- 包含所有Python源代码
- 包含配置文件和资源文件

## 文件使用指南

### 重构过程中
1. **参考文件**: 主要参考`MATLAB_Source/`中的文件
2. **测试数据**: 使用`NRAC_G023_Results/`中的.res文件进行测试
3. **文档参考**: 查看`Documentation/`中的文档了解功能和计划

### 开发顺序
1. 先阅读`功能分析文档.md`了解程序功能
2. 参考`重构计划.md`按步骤开发
3. 使用`NRAC_G023_Results/`中的数据进行测试
4. 对比`MATLAB_Source/`中的代码确保功能一致

## 注意事项

1. **NRAC_G023_Results文件夹必须保留**，这是程序的核心数据源
2. **MATLAB_Source文件夹中的文件是重构的主要参考**，需要仔细分析
3. **Documentation文件夹中的文档是重构的指导**，应经常查阅
4. **Archive文件夹中的文件暂时不需要**，但不要删除，可能作为参考
