## 程序功能

### 主要测试工况
1. **Bump测试** (parallel_travel) - 轮跳测试
2. **Roll测试** (opposite_travel/roll_angle) - 侧倾测试
3. **Static Load Lateral** - 侧向力测试
4. **Static Load Braking** - 制动力测试
5. **Static Load Acceleration** - 加速力测试

### 核心功能
- .res文件解析和K&C参数提取
- 数据计算和线性拟合
- 多类型图表绘制
- 性能指标计算
