# 快速开始

## 启动应用程序

### 方法1: 直接运行

```bash
python main.py
```

## 基本使用流程

### 1. 打开应用程序

启动后，您将看到主窗口界面。

### 2. 设置车辆参数

在主窗口的"车辆参数"面板中：
- 输入车辆基本信息
- 设置前后轴参数
- 配置其他相关参数

### 3. 加载数据文件

1. 点击"加载文件"按钮
2. 选择Adams仿真结果文件（.res文件）
3. 等待文件解析完成

### 4. 选择测试工况

在K&C分析Tab中选择要分析的测试工况：
- **Bump测试**: 轮跳测试
- **Roll测试**: 侧倾测试
- **Static Load**: 静态载荷测试

### 5. 查看结果

- 图表将自动显示在右侧
- 可以导出图表为图片
- 可以查看计算结果的数值

## 示例：Bump测试分析

1. **加载文件**: 选择 `NRAC_G023_parallel_travel.res`
2. **选择工况**: 点击"Bump测试"Tab
3. **选择参数**: 从下拉菜单选择要分析的参数（如toe_angle_left）
4. **查看图表**: 图表将显示参数随行程的变化
5. **查看结果**: 查看计算得到的刚度值等指标

## 快捷键

- `Ctrl+O`: 打开文件
- `Ctrl+S`: 保存配置
- `Ctrl+Q`: 退出应用程序
- `F5`: 刷新数据
